# AlGraphy Studio
**By Ahmad Al Hashmi**

AlGraphy Studio is a high-end digital design agency portfolio website featuring a modern, dark-themed user interface with immersive animations and smooth interactions.

## Overview

This project showcases a strategic design studio's capabilities in Strategy, Design, and Development. It utilizes advanced frontend techniques including smooth scrolling (Lenis), complex GSAP animations, and custom cursor text effects to create a premium user experience.

## Features

- **Immersive Hero Section**: Background video with cinematic text reveals and entrance animations.
- **Smooth Scrolling**: Implemented using [Lenis](https://github.com/studio-freight/lenis) for a fluid scroll experience.
- **Custom Cursor**: Interactive custom cursor that reacts to hover states and movement.
- **Interactive Services**: 3D-like card hover effects with parallax backgrounds.
- **Dynamic Portfolio**: Alternating project layouts with magnetic buttons and image reveals.
- **Video Modal**: Full-screen cinematic video player for the agency showreel.
- **Contact Modal**: Elegant full-screen overlay for project inquiries.
- **Responsive Design**: Fully optimized for various screen sizes.

## Technologies Used

- **HTML5**: Semantic markup.
- **CSS3**: Custom properties (variables), Flexbox, Grid, and advanced animations.
- **JavaScript (ES6+)**: Modular logic for UI interactions.
- **Libraries**:
  - [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform) for robust animations.
  - [Lenis](https://lenis.studiofreight.com/) for smooth scrolling.
- **Fonts**: Montserrat (Google Fonts).

## Project Structure

- `index.html`: Main structure and content.
- `style.css`: All visual styling, including responsive design and animations.
- `script.js`: Logic for animations, modal handling, scroll effects, and cursor behavior.
- `Assets/`: Directory containing images, videos, and GIFs.

## How to Run

1. Clone or download the repository.
2. Ensure the `Assets` folder contains the required media files.
3. Open `index.html` in any modern web browser.
