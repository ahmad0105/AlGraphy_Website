# AlGraphy Studio
**By Ahmad Al Hashmi**

AlGraphy Studio is a premium digital design agency platform featuring a cinematic frontend and a robust custom-built administrative dashboard.

## Architecture Overview

This project is a full-stack application organized into three main layers:
1. **Frontend (Landing Page)**: A high-performance interactive experience powered by PHP, GSAP, and Lenis.
2. **Dashboard (Admin Suite)**: A specialized administrative interface for managing clients, projects, services, and studio analytics.
3. **Backend (API Engine)**: A clean, OOP-based PHP backend (PSR-4 compliant) handling security, authentication, and database operations.

## Key Features

- **Cinematic UI**: Background video integration with custom GSAP reveal animations.
- **Unified Auth System**: Secure login/signup architecture synchronized between landing page and dashboard.
- **Resource Management**: Dynamic CRUD operations for Projects, Services, and Employees.
- **Studio Analytics**: Real-time project tracking, budget analysis, and team performance metrics.
- **Premium UX**: Custom magnetic cursor system and smooth scroll integration.

## Project Structure

- `index.php`: The main entry point for the landing page.
- `auth.html`: Unified authentication portal.
- `dashboard/`: Complete suite of administrative tools and statistics.
- `algraphybackend/`: Professional PHP backend source code.
- `database/`: SQL schema and database migration files.
- `Assets/`: Centralized directory for media assets (images, video, etc.).
- `css/` & `js/`: Core stylesheets and UI logic modules.

## Setup & Delivery

1. **Database**: Import the `.sql` file located in the `database/` folder.
2. **Environment**: This project is optimized for XAMPP/Apache (PHP 8.x+).
3. **URL**: Designed to run at `http://localhost/algraphy/`.
